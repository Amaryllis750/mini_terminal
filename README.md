# Mini Shell
## Description
This is a mini shell written in C. It is not POSIX compliant as I do not know POSIX specifications. 

## Features
- Run Commands with operators such as `|`, `>`, `;`, `>>`, `<`, `2>`
- Running of subshells (in instance of continuation)

## What my shell does not do
- Runs scripts
- Parse mathematical expressions
- Stores variables
